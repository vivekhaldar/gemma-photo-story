# Test photo example

These files were generated from a local vacation-photo folder with the default
`gemma4:26b` model. The source photos are intentionally not committed.

- `analysis.json` contains sanitized metadata, place lookup results, and
  per-photo visual analysis. Personal source paths and exact GPS/altitude values
  have been removed from this public example.
- `story.md` is Gemma's final narrative.
