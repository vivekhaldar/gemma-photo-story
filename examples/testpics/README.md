# Test photo example

These files were generated from `/Users/haldar/Downloads/testpics` with the
default local `gemma4:26b` model. The source photos are intentionally not
committed.

- `analysis.json` contains the extracted metadata, place lookup results, and
  per-photo visual analysis.
- `story.md` is Gemma's final narrative.
- `geocode-cache.json` makes reruns deterministic and avoids unnecessary public
  Nominatim requests.

The example contains exact GPS coordinates. Keep that privacy implication in
mind before making a fork or derivative repository public.
